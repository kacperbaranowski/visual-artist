# Visual Artist - Portfolio Template

A demo portfolio template for visual artists, photographers, and creators.

## Demo

This is a demo version. Full version available on the author's website.

## Tech Stack

- **[Astro](https://astro.build/)** - Web framework
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Styling
- **[GSAP](https://greensock.com/gsap/)** - Animations
- **[Lenis](https://lenis.darkroom.engineering/)** - Smooth scroll

## Features

- Responsive design
- Custom cursor
- Preloader
- Smooth scrolling
- GSAP animations
- Gallery lightbox
- Dynamic portfolio and exhibition pages
- Blog section

## Pages

- `/` - Home
- `/about` - About the artist
- `/portfolio` - Gallery
- `/exhibitions` - Exhibitions
- `/blog` - Blog
- `/contact` - Contact

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
